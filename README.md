# Uboontu Foundation

Uboontu Foundation is a modern, responsive web application built with Next.js. The foundation drives local community circularity, clean cities, worker dignity, and environmental sustainability initiatives.

## 🚀 Features

- **Dynamic Homepage:** Modern split-screen sections, circular progress indicators, and interactive initiative rows.
- **Featured Projects Stacking Layout:** Modern, interactive card stack layout that animates on scroll.
- **Get Involved Programs:** Comprehensive forms and guides for joining as a Volunteer or Partner.
- **Integrated Blog:** Responsive blog posts, single-column container layout, and customizable amber newsletter badge.
- **Donate Modal:** Highly polished multi-frequency donation modal that supports custom inputs.
- **Dark Mode Support:** Fully integrated dark theme matching Uboontu's curated green/amber brand identity.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **UI & Icons:** React, Lucide Icons
- **Styling:** CSS (Modular architecture)

## 💻 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
